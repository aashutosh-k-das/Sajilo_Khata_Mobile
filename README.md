# Sajilo Khata Mobile

Sajilo Khata Mobile is a mobile-first, offline-first digital ledger and billing app for MSMEs in Nepal. It helps users manage customers, track credit and payments, generate invoices, apply VAT calculations, and sync data with Firebase when online. It is built with Flutter for fast, reliable everyday business use.

## Features

- Customer management
- Credit and payment tracking
- Invoice generation
- VAT calculation support
- Offline-first experience
- Firebase sync when online
- Mobile-friendly business workflows
- Multi-language UI support
- SMS integration support

## Tech Stack

- Flutter / Dart
- Firebase
- JavaScript for app scaffolding and supporting utilities
- HTML, CSS, and JavaScript for the web delivery layer

## Project Structure

- `sajilo_khata_mobile/` — main Flutter app source
- `app.js` — supporting application logic
- `db.js` — data persistence and database helpers
- `locales.js` — localization strings
- `sparrow_sms.js` — SMS integration helpers
- `style.css` — styling
- `index.html` — app entry page
- `firebase.json` — Firebase configuration

## Getting Started

### Prerequisites

- Flutter SDK
- Dart SDK
- A code editor such as VS Code or Android Studio
- Firebase project configuration, if syncing features are enabled

### Installation

```bash
git clone https://github.com/aashutosh-k-das/Sajilo_Khata_Mobile.git
cd Sajilo_Khata_Mobile
flutter pub get
```

### Run the App

```bash
flutter run
```

If the app is configured for web:

```bash
flutter run -d chrome
```

## Configuration

If you plan to use Firebase syncing, update the Firebase settings in the project according to your environment.

## Build

```bash
flutter build apk
```

For web deployment:

```bash
flutter build web
```
